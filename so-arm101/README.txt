CS839 Final Project — Aswinkumar
Data Composition vs Architecture in Robot Imitation Learning

HOW TO VIEW
  REPORT: run `./start_report.sh` and open the printed URL, or open
          `final_report.html` directly in a browser. This is the research-style
          final deliverable with the required rubric sections.
  PPTX:  open `deck.pptx` in PowerPoint (or Keynote / LibreOffice). The GIFs
         play natively in slideshow mode; first frames are visible while editing.
  HTML:  open `deck.html` in Chrome/Firefox/Safari/Edge — ← / → to navigate,
         P to print to PDF (1920 × 1080, no margins, background graphics ON).

CONTENTS
  final_report.html — standalone project website / final report.
  start_report.sh   — local launcher for the final report website.
  deck.pptx       — 17-slide PowerPoint with each slide as a 1920×1080 image
                     plus animated GIFs overlaid at their original positions.
  deck.html       — same deck, HTML version (more responsive than PPTX).
  deck-stage.js   — custom <deck-stage> element handling navigation + print.
  plots/          — PNG charts and the marked-positions setup photo.
  gifs/           — robot rollout clips, dataset demos, dashboard recording.

NOTES
  All paths in deck.html are relative; keep the folder structure intact.
  GIFs animate inline in the browser. If exporting to PowerPoint, animations
  must be re-attached because PDF/print preserves only the first frame.
